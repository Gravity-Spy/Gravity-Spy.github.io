Train a Semantic Index.

This module uses keras to interface with the creation of a neural net and currently uses theano as the backend for doing the heavy gpu lifting.

The optimizer keras.optimizers.Adadelta

The loss function being optimized is softmax

Parameters:
data (str):
Pickle file containing training set data
model_folder (str, optional):
Defaults to model path to folder you would like to save the model
unknown_classes_labels (list, optional):
Defaults to [‘Whistle’, ‘Scratchy’] A list of classes to be considered as the unknown domain for which clustering will be performed once the knowledge from the known classes has been trained.
